# 🧮 Python Mini Calculator

Мой первый проект на Python — консольный калькулятор, поддерживающий основные арифметические операции.

## 🚀 Возможности

* ➕ Сложение (`+`)
* ➖ Вычитание (`-`)
* ✖️ Умножение (`*`)
* ➗ Деление (`/`)
* 🔢 Целочисленное деление (`//`)
* 💪 Возведение в степень (`**`)

## ▶️ Запуск

1. Убедитесь, что у вас установлен **Python 3**.
2. Склонируйте репозиторий или скачайте его архивом.
3. Откройте терминал в папке проекта.
4. Выполните команду:

```bash
python calculator.py
```

## 📋 Пример работы

```text
Введите первое число: 10
Введите второе число: 5
Введите операцию: +

Ответ: 15
```
